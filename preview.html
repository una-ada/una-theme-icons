when i was a wee kitten, i spent weeks drawing little guys in microsoft paint using the ball sprite from 3d pinball: space cadet as a template, but then i saved the file as a jpeg. this was the beginning of my villain arc.

the emoji in una-theme-icons are, of course, completely unrelated to this tragedy of days long past. most of the una-theme-icons use my palette; however, some of the emoji use the newer palette extended-a when necessary, e.g. for skin tones:🏻🏼🏽🏾🏿.

tangentially, regarding skin tones, if anyone has recommendations for better representing facial features on the person-* emojis, please let me know! you can open an issue on the git repo. this also goes for any other issue you spot with any of these emojis, but the legibility of those particularly bothers me.

one issue i currently have no intention to address is the bias for dark backgrounds, this holds true for all of una-theme-icons in general; you can test how these look with a light background by pressing `shift+D` on this page.

this page dynamically loads all unicode 17.0 emoji from a json file (parsed from emoji-test.txt) before running twemoji; it could take a few seconds to load. for a preview of non-emoji icons, see: ÁĆę.dev/log/tags
